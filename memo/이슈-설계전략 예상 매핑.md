## 이슈 → 설계전략 매핑

| 이슈 | 설명 | 주요 전략 | 보조 전략 |
|------|------|---------|---------|
| ISSUE-01 | 입장 집중 구간 처리 한계 | DS-02 (MQ), DS-04 (Priority Queue) | DS-05 (Eager Response), DS-09 (Bulkhead) |
| ISSUE-02 | 권한 갱신 외부 서버 종속 및 캐시 부재 | DS-03 (Cache) | DS-05 (Eager Response + 비동기 갱신), DS-10 (Circuit Breaker) |
| ISSUE-03 | 요청 유형 간 처리 우선순위 부재 | DS-04 (Priority Queue) | DS-06 (Throttling) |
| ISSUE-04 | DB 커넥션 풀 미격리 | DS-09 (Bulkhead) | DS-01 (MSA — 도메인별 DB 분리) |
| ISSUE-05 | 외부 서버 동기 호출 연쇄 지연 | DS-02 (MQ), DS-05 (Eager Response) | DS-03 (Cache — 외부 호출 결과 캐싱) |
| ISSUE-06 | 외부 서버 장애 격리 구조 부재 | DS-10 (Circuit Breaker / Fallback) | DS-09 (Bulkhead), DS-02 (MQ) |
| ISSUE-07 | 미팅 상태 Command/Query 모델 공유 충돌 | DS-08 (CQRS) | DS-01 (MSA — 읽기/쓰기 서비스 분리) |
| ISSUE-08 | 연계 시스템 증가 의존성 누적 | DS-11 (ACL), DS-01 (MSA) | DS-10 (Circuit Breaker) |
| ISSUE-09 | 예측 가능한 피크 트래픽 선제 대응 구조 부재 | DS-07 (Predictive Pre-warming) | DS-06 (Throttling), DS-03 (Cache) |
