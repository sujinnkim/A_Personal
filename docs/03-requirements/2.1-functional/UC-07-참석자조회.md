# UC-07. 회의 참석자 조회

## 개요

Knox 임직원 포탈 서버를 통해 회의 참석자 정보를 조회하여 반환한다.

## 행위자

- **주 행위자:** 미팅 웹포탈, 미팅 모바일 앱

## 사전 조건

- 조회 권한을 보유한 사용자가 로그인된 상태이다.

## 사후 조건

- 참석자 목록이 클라이언트에 반환된다.

## 기본 흐름

1. 사용자가 참석자 조회를 요청한다.
2. 포털 서버가 Knox 임직원 포탈 서버에 참석자 정보를 조회한다.
3. 조회 결과가 클라이언트에 반환된다.

## 예외 흐름

### Knox 서버 응답 지연 또는 장애

1. Knox 임직원 포탈 서버가 응답을 지연하거나 오류를 반환한다.
2. 포털 서버가 참석자 조회 실패를 클라이언트에 반환한다.
