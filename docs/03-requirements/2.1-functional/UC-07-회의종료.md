# UC-07. 회의 종료

## 개요

진행 중인 회의를 종료하고 WC/VC/AC 외부 서버에 종료를 전파한다.

## 행위자

- **주 행위자:** 미팅 웹포탈, 미팅 모바일 앱

## 사전 조건

- 회의가 진행 중인 상태이다.
- 종료 권한을 보유한 사용자가 로그인된 상태이다.

## 사후 조건

- 회의 상태가 DB에 '종료'로 업데이트된다.
- 외부 서버(WC/VC/AC)에 종료가 전파된다.

## 기본 흐름

1. 사용자가 회의 종료를 요청한다.
2. 포털 server-api가 WC/VC/AC 서버에 종료를 전파한다.
3. 회의 상태가 DB에 '종료'로 업데이트된다.

## 예외 흐름

### 외부 서버 종료 전파 실패

1. WC/VC/AC 서버에 종료 전파 요청이 실패한다.
2. 포털 DB는 종료 처리되었으나 외부 서버 상태와 불일치가 발생한다.
